# 샬롯의 테이블
비밀스런 스토리를 가진 주인공이 최고의 셰프로 성장해가는 스토리를 가진 게임으로써 게임 방식은 Match-3 퍼즐 형식입니다.
퍼즐을 클리어하여 얻은 재화로 요리를 하고 테이블 세팅과 레스토랑을 꾸밀 수 있습니다.
